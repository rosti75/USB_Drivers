This is the USB driver for Tagsys USB RFID Readers.

Supported Readers :

	- L-P101
	- Medio P101 Ethernet
	- Medio P101 Wifi
	- Medio L100/L200
	- Medio L400
	- Medio S004 EVM
	- SRU400
	- Medio P200u
	- Medio P200x
	- Medio L40

Supported OSes :
 
	- Windows Server 2008 R2
	- Windows 7
	- Windows 7 x64
	- Windows Server 2008
	- Windows Server 2008 x64
	- Windows Vista
	- Windows Vista x64
	- Windows XP
	- Windows XP x64
	- Windows 2000
	- Windows Server 2003
	- Windows Server 2003 x64

Though not WHQL certified, this driver has been carefully tested with the above operating systems. You can safely go through the install process even if Windows warns you about the driver not being WHQL certified.
